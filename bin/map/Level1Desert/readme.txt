Informationen zum Erstellen neuer Maps:

Mapgröße: nach Belieben.
Kachelgröße: 59x59 pixel!

(Da Tileset_Desert auf ein 59x59 Raster ausgelegt)


Attribute: boolean walkable, buildable;

Kachelebene die den path enthält, erhält die Attribute: walkable=true; buildable=false;
Kachelebene für Baurechte, erhält Attribute: walkable=false; buildable=true;
